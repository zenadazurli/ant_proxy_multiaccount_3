# AntAutoSurf Multi-Account Bot

Bot multi-account per AntAutoSurf.com con Proxy Manager automatico potenziato.

## 🚀 Caratteristiche

- ✅ Multi-account in parallelo (fino a 5+ contemporaneamente)
- ✅ Proxy Manager con 20+ fonti di proxy
- ✅ Creazione account automatica
- ✅ Captcha risolto automaticamente
- ✅ Riavvio automatico
- ✅ Headless mode
- ✅ Resiliente ai proxy morti
